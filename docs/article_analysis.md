## 高互动文章特征分析

### TOP3 高互动文章

**1. a2 — AI使用水平10个等级（★★★★★）**
- 标题：观察了三年，我把所有人用AI的水平分成了10个等级
- 结构：场景故事 → 四个维度 → 十个等级逐步展开 → 自我定位引导
- 亮点：
  - 等级/分级框架让读者产生"我在哪"的自我定位冲动
  - 游戏类比（打怪升级）降低理解门槛
  - 每个等级都有具体特征描述，可操作
  - 结尾引导读者分享给朋友（社交传播）

**2. a6 — GPT-image-2黑暗森林（★★★★★）**
- 标题：因为GPT-image-2，整个互联网都变成了巨大的黑暗森林
- 结构：反差开头 → 具体现象 → 三体类比 → 三条公理 → 升华
- 亮点：
  - 第一句话制造认知冲击（"黑暗森林"）
  - 跨领域类比（三体猜疑链）极具传播力
  - 金句密度极高（"当造假成本趋近于零，信任成本就趋近于无穷大"）
  - 从技术现象上升到哲学/社会层面

**3. a11 — Claude Code保姆教程（★★★★★）**

  - 反复测试确保稳定性（"折腾了五六台电脑"）

## 语言艺术提炼

### 卡兹克式写作特征

1. **口语化叙事**："说实话"、"坦率的讲"、"一声长叹"
2. **吐槽真实感**："不学Claude好的方面，净学这方面"
3. **数据驱动**：跑分、价格、参数——具体数字
4. **跨领域类比**：游戏（只狼、金铲铲、全面战争）、三体、马具
5. **金句密度**：每段都有可截图传播的话
6. **真实感铺垫**："我自己已经内测使用了快1个月了"
7. **产品链接**：GitHub、官网——方便读者直接使用

### 高互动标题公式

1. **数字型**：分享6个我觉得应该必装的Skills
2. **反差型**：因为GPT-image-2，整个互联网都变成了巨大的黑暗森林
3. **等级型**：观察了三年，我把所有人用AI的水平分成了10个等级
4. **实测型**：实测DeepSeek V4，为国产化而生
5. **时效型**：豆包输入法的Mac版，终于正式上线了

### 高互动开头模式

1. **反差/震撼开头**："整个互联网都变成了巨大的黑暗森林"
2. **场景观察开头**："上周末跟几个老朋友吃饭，吃到一半就聊到AI了"
3. **等待释放开头**："终于正式上线了。我等这一天，等了好久好久。"
4. **热度引入开头**："最近这个词实在是太火了。刷推刷到，朋友圈刷到"
