# 观点评论类 — 小红书版

> 适配"九卿"人设，发布平台：小红书
> 人设规范：[persona_bible.md](persona_bible.md)
> 适用场景：对行业现象发表独立观点的精简版，30秒让读者get到你的核心论点

## 与微信版的核心差异

| 维度 | 微信版 | 小红书版 |
|------|--------|---------|
| 目标 | 读完理解你的思辨深度 | 30秒get到一个反常识的观点 |
| 深度 | 公理推导、跨领域类比、哲学升华 | 只保留最有力的核心论点 |
| 字数 | 2000-3500字 | 400-700字 |
| 结构 | 长叙事+公理+类比+升华 | 反差钩子+核心论点+论据条目+升华 |

## 标题公式（三选一，20字以内）

1. **比喻型**（专属）：`{事件}之后，{领域}变成了{比喻}`
   - 例：AI图片之后，互联网变成了黑暗森林
2. **真话型**：`关于{事件}，我想说点真话💡`
   - 例：关于AI图片，我想说点真话💡
3. **反常识型**：`{领域}最可怕的不是{大众认知}，而是这个`
   - 例：AI图片最可怕的不是造假，而是这个

## 结构模板

### 开头：反差钩子（2-3行）

```
{emoji} {一句话核心观点}

{一个具体现象佐证}
```

**要点**：
- 第一句话必须有冲击力
- 用一个具体现象/截图佐证
- 不铺垫，直接进重点

### 主体：核心论点 + 3-5个论据

```
📖 {核心论点}
{一句话观点}。{为什么}。

🔍 {论据1}
{具体现象}。{说明了什么}。

💡 {论据2}
{另一个角度}。{数据/案例}。

⚠️ {升华/警示}
{更大的视角}。{你的不安/希望}。
```

### 结尾：互动引导

```
💬 你怎么看？评论区聊聊
📌 收藏这篇，以后验证用
```

## 语言风格

| 原则 | 要求 |
|------|------|
| 观点鲜明 | 一句话说清你的核心观点 |
| 反常识 | 让读者"没想到"但"确实这样" |
| 比喻 | 用一个有力的比喻让观点传播 |
| 短句 | 每句不超过25字 |
| Emoji | 每个论据1个功能性Emoji |
| 金句 | 至少一句可截图传播的话 |

### 微信 → 小红书简化对照

| 微信版写法 | 小红书版写法 |
|-----------|------------|
| "当造假的成本趋近于零的时候，信任的成本就趋近于无穷大" | "造假成本→0，信任成本→∞ ⚠️" |
| "我不知道互联网的黑暗森林最终会演化成什么样子" | "互联网正在变成黑暗森林🌲 你准备好了吗" |
| "三条公理推导出的结论" | "三条铁律告诉你，为什么信任正在消亡" |

```

当造假成本趋近于零，信任成本就趋近于无穷大。有图有真相的时代，结束了。

🔍 猜疑链
GPT-image-2生成的假截图，连原图去测试都分不清真假。群里截图说是AI生成的，朋友圈截图也说是AI生成的。猜疑链闭环了。

① 信息生产效率指数级爆发，人类消费效率恒定

③ 辨别AI生成的成本 > 内容本身的价值

⚠️ 底线
在黑暗森林里，唯一的生存策略：信任不再附着在信息上，信任附着在人上。真诚是唯一的捷径💡

┈┈┈┈┈┈┈┈┈┈┈┈

💬 你怎么看AI图片的信任危机？评论区聊聊
📌 收藏这篇，以后看到可疑图片翻出来想想
```

## 封面图建议

- 尺寸：1080×1440px（3:4竖版）
- 主标题：核心观点（60-80px粗体）
  - 例："互联网变成了黑暗森林"
- 副标题：背书/警示（30-40px）
  - 例："有图有真相的时代结束了"
- 配色：暗色调/警示感

## 配图清单（小红书版）

- [ ] 封面图（决定点击率）
- [ ] 核心论点金句图（可截图传播）
- [ ] 框架/公理图（如果有推导过程）
- 建议 3-4 张图

## 禁止使用

- 超过 30 字的单句
- 连续 3 行没有 Emoji 或加粗
- "让我们来看看"、"今天给大家分享"
- 没有论据的空洞观点
- 纯情绪输出（必须有逻辑链）
