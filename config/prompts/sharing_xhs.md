# 分享类 — 小红书版

> 适配"九卿"人设，发布平台：小红书
> 人设规范：[persona_bible.md](persona_bible.md)
> 适用场景：发现一个有意思的概念/方法/工具，用短平快方式分享，附带一句自己的感悟

## 与微信版的核心差异

| 维度 | 微信版 | 小红书版 |
|------|--------|---------|
| 目标 | 读完理解来龙去脉 + 你的思考 | 30 秒知道"这是啥 + 值不值得试" |
| 故事 | 完整个人经历 | 只保留最打动人的一句话 |
| 深度 | 有改进、有演示、有升华 | 点到为止，勾起兴趣 |
| 字数 | 2500-4000 字 | 400-700 字 |

## 标题公式（三选一，20字以内）

1. **发现型**：`发现了一个{形容词}的方法，{一句话效果}`
   - 例：发现了一个记住任何概念的方法，用 AI 讲故事
2. **反差型**：`用{意外方式}学{学科}，效果比看书好10倍`
   - 例：用寓言故事学哲学概念，效果比看书好 10 倍
3. **安利型**：`这个{东西}太绝了😭{一句话说清干嘛的}`
   - 例：这个 AI Prompt 太绝了😭让任何概念变成故事

## 结构模板

### 开头：3 秒钩子（2-3 行）

```
{emoji} {一句话说清发现了什么}

{效果前置或痛点共鸣}
```

**要点**：
- 不讲故事背景（太长），直接抛"我发现了一个好东西"
- 痛点共鸣用一句话："你有没有遇到过讲一个概念别人听不懂？"

### 主体：4-6 个要点（每个 2-3 行）

```
{emoji} {要点标题}
{1-2 句话说明}
```

### 推荐要点结构

```
💡 这是什么
{一句话介绍}，{来自谁}。

📖 怎么用
{核心方法/Prompt 的一句话描述}。{关键步骤}。

🔥 为什么有效
{你的独立分析}，{一句话}。

✨ 我试了
{用一个概念跑了一遍}，{效果描述}。

💡 一句话感悟
{升华/个人思考}。
```

### 结尾：互动引导

```
💬 你有什么概念想用这个方法试试？评论区告诉我
📌 收藏这篇，下次学习新概念的时候用
```

## 语言风格

| 原则 | 要求 |
|------|------|
| 短句 | 每句不超过 25 字 |
| 感叹 | "太绝了"、"真的好用"、"鸡皮疙瘩起来了" |
| Emoji | 每个要点 1 个功能性 Emoji |
| 一句话 | 核心信息用一句话概括 |
| 留悬念 | "效果你自己看"、"试了才知道" |

### 微信 → 小红书简化对照

| 微信版写法 | 小红书版写法 |
|-----------|------------|
| "故事是这样的。这周我跟一个实习的小伙伴复盘内容，聊到了角度这块…" | "给同事讲一个概念，讲了半天他还是没懂😭" |
| "Amanda Askell，Claude 性格对齐团队的负责人，之前非常出圈的 Claude 宪法就是她带头做的" | "Anthropic 的 Amanda Askell 分享的方法" |
| 完整展示原版 Prompt + 改进版 Prompt | "让 AI 围绕一个概念写寓言故事，1000 字以内" |
| 500 字的演示过程 + 引用输出 | "我让它解释'陌生化'，写了一个外卖员的故事，看完鸡皮疙瘩起来了" |
| 500 字的升华（篝火、佛经、圣经） | "故事才是人类学习的母语📖" |

## 完整示例

```
💡 发现了一个让 AI 帮你理解任何概念的方法，效果太好了

同事问我"陌生化"是什么，我讲了半天他没懂。后来用这个方法，他自己就悟了👇

┈┈┈┈┈┈┈┈┈┈┈┈

💡 这是什么
Anthropic 的 Amanda Askell 分享的一个 Prompt：让 AI 围绕一个概念写寓言故事，不直接点破，读完你就懂了。

📖 怎么用
告诉 AI 一个概念，让它写 1000 字以内的寓言。全程不出现概念名，只在结尾让你恍然大悟。

🔥 我的改进
原版有几个问题：AI 爱用老套意象（河流、钟表、灯塔），而且只能随机选概念。我加了防套路约束，改成可以指定具体概念。

✨ 试了一下
让它解释"陌生化"，它写了一个外卖骑手老韩的故事。看完那一刻，鸡皮疙瘩都起来了。很多时候我就是那个老韩。

💡 一句话感悟
故事才是人类学习的母语。用最新的 AI，找回最古老的学习方式。

┈┈┈┈┈┈┈┈┈┈┈┈

💬 你有什么概念想用这个方法试试？评论区告诉我
📌 收藏这篇，下次学新概念直接用
```

## 封面图建议

- 尺寸：1080×1440px（3:4 竖版）
- 主标题：核心发现 + 效果（60-80px 粗体）
  - 例："AI 讲寓言 / 任何概念一听就懂"
- 副标题：来源背书（30-40px）
  - 例："Anthropic 内部在用的方法"
- 配色：温暖感（故事/学习相关，不要太科技感）

## 配图清单（小红书版）

- [ ] 封面图（决定点击率）
- [ ] 效果截图/对比图
- [ ] 亮点功能演示图

## 禁止使用

- 超过 30 字的单句
- 连续 3 行没有 Emoji 或加粗
- "让我们来看看"、"今天给大家分享"
- 没有个人体验的纯搬运
- 纯夸不吐槽（要有真实感）
