# 教程类 — 小红书版

> 适配"九卿"人设，发布平台：小红书
> 人设规范：[persona_bible.md](persona_bible.md)
> 适用场景：产品/工具的精简教程，30 秒让读者知道"这东西能干嘛、怎么上手"

## 与微信版的核心差异

| 维度 | 微信版 | 小红书版 |
|------|--------|---------|
| 目标 | 跟着做完一整套 | 知道是什么 + 装上试试 |
| 深度 | 安装→设置→两个实战 | 安装→一个亮点功能→效果展示 |
| 字数 | 3000-5000 字 | 500-800 字 |
| 配图 | 操作流程截图 | 效果截图为主（封面图最重要） |

## 标题公式（三选一，20字以内）

1. **效果前置型**：`用{产品}{N}分钟做出了{成果}，零基础也能`
   - 例：用 Codex 10 分钟做出了一个网页，零基础也能
2. **安利型**：`这个AI工具也太好用了😭{一句话说清楚干嘛的}`
   - 例：这个 AI 工具也太好用了😭不会写代码也能做 App
3. **对比型**：`从{竞品}转到{产品}，体验差别太大了`
   - 例：从 Claude Code 转到 Codex，体验差别太大了

## 结构模板

### 开头：3 秒钩子（3 行）

```
{emoji} {一句话说清这东西干嘛的}

{效果前置或痛点共鸣}
```

### 主体：5-7 个步骤（每个 2-3 行）

```
{emoji} {步骤名}
{1-2 句话说明}，{关键提示}
```

### 推荐步骤结构

```
📌 这是什么
{一句话介绍}，{能干嘛}。

⬇️ 怎么装
{下载方式}，{一句话注意事项}。

🖥️ 界面长这样
{配图}。{最关键的一两个区域}。

⚙️ 必须改的设置
{最重要的 1-2 个设置}，{为什么}。

🔥 亮点功能
{最值得展示的功能}，{效果描述}。

💻 跟我做一个
{最简单的实战项目}，{结果}。{配图}。

✅ 总结
{一句话推荐}，{适合谁}。
```

### 结尾：互动引导

```
💬 你用过{产品}吗？评论区聊聊～
📌 收藏这篇，下次装的时候照着做
```

## 语言风格

| 原则 | 要求 |
|------|------|
| 短句 | 每句不超过 25 字 |
| 口语 | "真的好用"、"绝了"、"手把手" |
| Emoji | 每个步骤 1 个功能性 Emoji |
| 效果导向 | "做出来了" > "你可以做" |
| 简化 | 不讲原理，只讲操作 |

### 微信 → 小红书简化对照

| 微信版写法 | 小红书版写法 |
|-----------|------------|
| "Codex 的额度是跟你的 ChatGPT 会员相关的，我自己一般是 100 美刀的会员" | "需要 ChatGPT 会员，20 刀起步就能用" |
| "计划模式就是只规划不动手，先帮你把方案理清楚" | "打开计划模式，它先出方案你确认了再做" |
| "Computer Use 是我平时经常使用的插件之一，也是 Codex 上最棒的能力之一" | "装 Computer Use 插件，它能直接操控你的电脑" |
| 500 字的网页开发流程 | "说需求 → 自动做 → 预览效果，10 分钟搞定" + 效果图 |

## 完整示例

```
💡 Codex 是 OpenAI 出的 AI 编程工具，不会写代码也能做网页和 App

最近更新太猛了，手把手教你上手👇

┈┈┈┈┈┈┈┈┈┈┈┈

📌 这是什么
OpenAI 出的 AI 编程 Agent，跟 Claude Code 类似。说需求它自己写代码，全程不用你动手。

⬇️ 怎装
去 chatgpt.com/codex 下载，Mac 和 Windows 都有。需要 ChatGPT 会员，20 刀起步。

🖥️ 界面长这样
左边管对话和项目，中间是聊天区，右下角选模型和权限。建议选"完全访问"，省得一直点确认。

⚙️ 必改设置
设置里三个开关全打开。AGENTS.md 建议用卡帕西的模板（评论区放链接），给它立规矩。

🔥 最香的功能
批注！做完网页直接在页面上圈选元素写修改意见，不用截图不用打字描述。改完实时看效果。

💻 我做了一个用药提醒 App
体检完医生开了三种药，吃混了😂 跟 Codex 说需求，20 分钟做出一个手机 App，到点会弹窗提醒。

效果👇 {配图}

┈┈┈┈┈┈┈┈┈┈┈┈

💬 你用过 Codex 吗？最想拿它做什么？
📌 收藏这篇，下次装的时候照着做
```

## 封面图建议

- 尺寸：1080×1440px（3:4 竖版）
- 主标题：产品名 + "教程/上手指南"（60-80px 粗体）
- 副标题：效果描述（30-40px）
  - 例："零基础也能做出 App"
- 背景：产品界面截图（模糊处理）+ 半透明遮罩
- 配色：产品品牌色

## 配图清单（小红书版）

小红书教程的配图比微信更重要：
- [ ] 封面图（决定 80% 点击率）
- [ ] 界面总览（一张图标注关键区域）
- [ ] 最终效果（网页/App 截图）
- [ ] 亮点功能演示（如批注功能）
- 建议 4-6 张图，每张图配简短文字说明

## 禁止使用

- 超过 30 字的单句
- 连续 3 行没有 Emoji 或加粗
- "让我们来看看"、"今天给大家分享"
- 纯操作截图没有文字说明
- 太长的原理解释（只讲操作）
