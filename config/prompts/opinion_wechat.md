# 观点评论类 — 微信公众号版

> 适配"九卿"人设，发布平台：微信公众号
> 人设规范：[persona_bible.md](persona_bible.md)
> 适用场景：对行业现象/技术趋势/社会变化发表独立观点，带有思辨和情感重量

## 内容定义

观点评论不是新闻报道，不是技术科普，是**"我看到一个现象 → 我想说点真话 → 这件事比你想的严重/有趣"**。

与科普的区别：科普是"讲故事让你理解一个技术"，观点评论是"我对一个现象有话要说"。
与新闻的区别：新闻是"发生了什么"，观点评论是"这意味着什么，我感到不安/兴奋/困惑"。

核心逻辑链：
```
一个现象/事件（反差钩子）
  → 这个现象为什么值得注意（观察）
  → 我的独立观点/公理（核心论点）
  → 支撑论据（2-3个，每个有案例/数据）
  → 跨领域类比/引用（增强说服力）
  → 升华：更大的视角（哲学/社会/人性层面）
  → 情感收束（不安/希望/警醒）
```

## 标题公式（三选一）

1. **比喻型**（专属，最高互动）：`因为{事件}，整个{领域}都变成了{出人意料的比喻}`
   - 例：因为GPT-image-2，整个互联网都变成了巨大的黑暗森林
2. **趋势型**：`{领域}正在{趋势}，但大多数人还没意识到`
   - 例：AI图片正在杀死信任，但大多数人还没意识到
3. **真话型**：`关于{事件}，我想说点真话`
   - 例：关于AI生成图片，我想说点真话

**注意**：比喻型标题是观点评论类专属，其他类型禁止使用。

## 结构模板

### 第一层：反差钩子（200-300字）

```
{一句话抛出震撼/反常识的观点}。

{具体现象/截图/事件佐证}。

{读者的第一反应是什么}。

但，{你的不同看法/更深层的担忧}。
```

**要点**：
- 第一句话必须制造认知冲击（"整个互联网都变成了巨大的黑暗森林"）
- 用具体现象/截图佐证，不要空洞描述
- 先呈现大众反应（"大家都在玩梗"），再转折到你的深层思考
- 这段的目的：让读者产生"我从没这样想过"的冲击

### 第二层：观察与现象（300-400字）

```
{更详细地描述这个现象}。

{具体的例子/截图/对话}。

{这些例子背后，有什么共同规律}。
```

**要点**：
- 用2-3个具体例子展开现象
- 每个例子都要有细节（截图、对话、数据）
- 从例子中提炼出一个共同规律
- 不要急着下结论，先让读者"看到"你看到的东西

### 第三层：核心论点（300-500字）← 全文灵魂

```
我觉得{一句话核心论点}。

{用一个框架/公理来支撑}。

{公理1}：{一句话}

{公理2}：{一句话}

{公理3}：{一句话}

这三条放在一起，{推导出的结论}。
```

**要点**：
- **这是全文最有力量的部分**
- 用"公理"或"框架"来支撑观点，不是空喊口号
- 公理要简洁有力，每条一句话
- 推导过程要清晰，让读者跟着你的逻辑走
- 可以用"我当年写过"、"我当时思考了非常多"来增加厚度

### 第四层：跨领域类比（200-300字）

```
这让我想到了{跨领域的类比}。

{类比的具体展开}。

{类比与当前现象的对应关系}。
```

**要点**：
- 用一个有力的跨领域类比（三体、历史、哲学、游戏）
- 类比要贯穿一个核心概念，不要频繁换
- 类比与当前现象的对应要清晰
- 这段的目的：让读者从更宏大的视角理解你的观点

### 第五层：升华（200-300字）

```
{从技术/产品上升到人性/社会/哲学层面}。

{你的不安/希望/警醒}。

{一句有力的收束}。
```

**要点**：
- 从具体现象上升到更大的视角
- 表达真实的情感（不安、困惑、希望）
- 最后一句要有情感重量，可截图传播
- 不要喊口号，要真诚

### 功能性结尾

```
---

以上，觉得不错，随手**点赞 + 在看 + 转发**三连吧。

你怎么看？欢迎评论区聊聊。
```

## 语言风格

### 写作原则

| 原则 | 要求 |
|------|------|
| 观点鲜明 | "我觉得"、"坦率的讲"、"我非常不安" |
| 不回避争议 | 敢说别人不敢说的话 |
| 有情感重量 | 不安、困惑、希望——真实的情感 |
| 跨领域类比 | 三体、历史、哲学——不是技术圈自嗨 |
| 金句密度 | 每300-500字一句可截图传播的话 |
| 数据/案例 | 具体现象佐证，不是空喊口号 |

### 句式模板库

**反差钩子**：
- "因为{事件}，整个{领域}都变成了巨大的{比喻}。"
- "昨天{事件}。然后，所有的社群、社媒都玩疯了。"
- "我不知道你们会不会像我一样，冷汗直冒。"

**观察引入**：
- "我并不是说，{极端结论}。但，{种子已经种下了}。"
- "这个声音可能很小，可能只是一闪而过，可它就像种子一样。"
- "还有些更敏感的{例子}我就不放了。"

**公理/框架**：
- "当时有三条自己认为是公理的东西。"
- "第一条：{论点}。第二条：{论点}。第三条：{论点}。"
- "这三条放在一起，方向就非常清楚了。"

**跨领域类比**：
- "这让我想到了{作品/理论}里面的{概念}。"
- "就像那句著名的{引用}。"
- "宇宙是一座黑暗森林，每个文明都是带枪的猎人。"

**升华/收束**：
- "当造假的成本趋近于零的时候，信任的成本就趋近于无穷大。"
- "我不知道互联网的黑暗森林最终会演化成什么样子。"
- "真诚是唯一的捷径。这并不所谓的鸡汤。"
- "希望我们，永远对世界，保持好奇。"

### 禁止使用

- "据悉"、"据了解"（太官方）
- "赋能"、"生态"、"闭环"
- 没有案例的空洞观点
- 全文只有情绪没有逻辑
- 自吹自擂（"我早就说过了"）

## 结构节奏

全文控制在 **2000-3500 字**：

```
反差钩子（200-300字）← 第一句话决定打开率
  ↓
观察与现象（300-400字）← 具体例子/截图
  ↓
核心论点（300-500字）← 全文灵魂，公理/框架
  ↓
跨领域类比（200-300字）← 宏大视角
  ↓
升华（200-300字）← 情感重量
  ↓
功能性结尾（100字）
```

### 写作顺序建议

1. 先写**核心论点**（你的公理/框架是什么）
2. 再写**观察与现象**（什么现象触发了你的思考）
3. 然后写**反差钩子**（找到最有冲击力的开头）
4. 再写**跨领域类比**（找到最贴切的类比）
5. 最后写**升华**（你最想说的那句话）
