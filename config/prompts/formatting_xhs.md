# 小红书笔记排版格式规范

> 继承通用规范 [formatting.md](formatting.md)，本文件仅定义小红书特有参数。
> 适用于小红书图文笔记（300-800字）的排版设计。

## 核心差异

小红书与微信公众号的根本区别：

| 维度 | 微信公众号 | 小红书 |
|------|-----------|--------|
| 阅读场景 | 深度阅读 | 碎片化扫读 |
| 字数 | 1000-3000字 | 300-800字 |
| 排版工具 | 编辑器（富文本/Markdown） | 纯文本 + Emoji |
| 视觉依赖 | 低（文字为主） | 高（封面图决定 80% 点击率） |
| Emoji 角色 | 装饰 | **功能性元素**（结构标记、情绪、分隔） |

## 三行原则（最重要）

小红书是手机阅读，一屏约 8-10 行。超过 3 行的段落会形成"文字墙"，用户本能跳过。

- 每段控制在 **2-3 行**（约 40-60 字）
- 超过 3 行必须找逻辑断点拆分
- 段落之间用空行分隔
- 如果确实需要更长段落，用 Emoji 或加粗打断视觉连续性

### 好坏对比

```
坏：
今天给大家分享一个我用了很久的护肤方法其实很简单就是每天早晚各用一次维C精华然后搭配烟酰胺的面霜坚持用了三个月之后皮肤真的亮了很多而且痘印也淡了不少关键是不贵两个加起来才一百多块钱。

好：
分享一个用了 3 个月的护肤搭配，皮肤真的亮了很多✨

方法很简单：早晚各一次维C精华 + 烟酰胺面霜。

坚持 3 个月，痘印淡了、肤色均匀了。关键是两个加起来才 100 多块💰
```

## 总分总结构

每篇笔记的宏观结构：

1. **总（开头）**：2-3 行概括全文核心价值，告诉用户"看完能得到什么"
2. **分（主体）**：分点展开，每个点聚焦一个信息
3. **总（结尾）**：总结要点 + 互动引导

好处：即使用户只看了开头和结尾，也能获取核心信息。

## Emoji 使用规范

### 三大功能

#### 功能一：结构标记

用 Emoji 替代纯文字编号，视觉更突出：

| 场景 | 推荐 Emoji | 示例 |
|------|-----------|------|
| 步骤/教程 | ✅ 📌 💡 | ✅ 第一步：清洁面部 |
| 产品推荐 | 🌟 ⭐ 💎 | 🌟 推荐一：XX 面霜 |
| 注意事项 | ⚠️ ❌ 🚫 | ⚠️ 注意：敏感肌慎用 |
| 优点 | ✔️ 👍 💯 | ✔️ 持妆力很好 |
| 缺点 | ❌ 👎 💔 | ❌ 质地偏厚重 |
| 总结 | 📝 📋 🔑 | 📝 总结：适合油皮 |

#### 功能二：情绪表达

| 情绪 | Emoji | 使用场景 |
|------|-------|---------|
| 惊喜 | 😱🤯✨ | 发现好东西时 |
| 推荐 | 💕🫶❤️‍🔥 | 真心推荐时 |
| 吐槽 | 😭💀🤡 | 自嘲或吐槽时 |
| 提醒 | 👀📢‼️ | 强调重点时 |
| 省钱 | 💰🤑💸 | 性价比相关时 |

#### 功能三：视觉分隔

分隔线示例：

```
🔸🔸🔸🔸🔸🔸🔸🔸
┈┈┈┈┈┈┈┈┈┈┈
✦ ✦ ✦
```

### 用量控制

- 标题：0-2 个 Emoji，放在末尾或关键词旁
- 正文：每段 0-1 个，全文 **8-15 个**
- 密度：**不超过总字数的 5%**
- 同一篇笔记 Emoji 风格统一（圆形🔵🟢🔴 或方形🟦🟩🟥，不混用）

### 禁止的错误

```
坏：💕✨🌈今天给大家推荐🤩一款超级好用的面霜😍💖真的太绝了🔥🔥🔥姐妹们快冲💪💪
    ↑ Emoji 多到喧宾夺主

坏：今天去医院看了一下皮肤问题 🎉
    ↑ 情绪不对，容易误解

坏：⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐
    ↑ 超过 5 个重复 Emoji 像刷屏
```

## 关键词标记

- 重点词用【】包裹：【控油】【敏感肌】【性价比】
- 适用于产品名、核心概念、关键数据
- 每段最多 1-2 个【】标记，过多则失去强调效果

## 封面图设计

### 尺寸比例

| 类型 | 尺寸 | 比例 | 说明 |
|------|------|------|------|
| **竖版封面（推荐）** | 1080×1440px | **3:4** | 信息流中占屏比最大，点击率最高 |
| 横版封面 | 1080×810px | 4:3 | 适合对比图、场景图 |
| 正方形封面 | 1080×1080px | 1:1 | 适合产品展示 |

### 封面文字

- 主标题：**60-80px**，粗体/黑体，醒目突出
- 副标题：30-40px
- 避免过细字体（小屏阅读困难）
- 文字放在安全区域（避开顶部状态栏和底部操作栏）

### 封面设计原则

- 3 秒内传递核心价值
- 数字/对比制造好奇："5 个误区"、"3 个月对比"
- 配色吸引眼球但不刺眼
- 文字不超过 3 行

## 不同内容类型的排版要点

### 干货教程类

```
标题（含关键词）
开头钩子（2-3 行）
📌 要点一：XXX
   具体说明（2-3 行）
📌 要点二：XXX
   具体说明（2-3 行）
📌 要点三：XXX
   具体说明（2-3 行）
💡 总结
一句话总结 + 引导收藏
```

排版关键：**结构感 > 一切**。用编号和 Emoji 让每个要点一目了然。

### 好物种草类

```
标题（产品名 + 核心卖点）
场景引入（痛点共鸣，2-3 行）
🌟 卖点一：XXX
   体感描述
🌟 卖点二：XXX
   体感描述
使用效果（带数据或 before/after）
💡 适合人群 + 购买建议
```

排版关键：**图文搭配**。每个卖点最好配一张细节图。

### 个人故事类

```
标题（悬念/结果前置）
故事开头（背景 + 冲突）
故事发展（2-3 个段落）
转折点
收获/感悟
对读者的建议
```

排版关键：**自然段落为主**。像"朋友发的长消息"，太格式化反而不真实。

### 清单合集类

```
标题（数字 + 品类）
一句话说明清单价值
① XX名称 - 推荐理由（1 行）
② XX名称 - 推荐理由（1 行）
③ XX名称 - 推荐理由（1 行）
...
使用建议 + 引导收藏
```

排版关键：**格式高度统一**。每条结构必须一致，形成阅读节奏。

## 完整排版模板示例

```
油皮护肤的 5 个误区，踩过 3 个以上说明你一直在走弯路😭

控油这件事，我走了整整 2 年弯路。现在终于搞明白了，把最重要的 5 个误区整理出来👇

┈┈┈┈┈┈┈┈┈┈┈┈

❌ 误区一：疯狂洗脸就能控油
一天洗 3 次脸不会让你变干净，只会让皮肤觉得"太干了"从而分泌更多油脂。建议早晚各一次就够了。

❌ 误区二：油皮不需要保湿
油 ≠ 水分够。很多油皮其实是"外油内干"，越不保湿越出油。选择清爽型的保湿凝胶就行。

❌ 误区三：控油产品越强越好
酒精含量高的控油产品短期有效，长期会破坏皮肤屏障。选温和的水杨酸类控油产品。

┈┈┈┈┈┈┈┈┈┈┈┈

✅ 正确的油皮护肤顺序
温和洁面 → 补水精华 → 清爽保湿 → 防晒

就这么简单，不需要叠加 7、8 层。少即是多。

觉得有用的话点个收藏📌 下次护肤的时候翻出来对照一下～
```
