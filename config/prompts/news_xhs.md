# 新闻类 — 小红书版

> 适配"九卿"人设，发布平台：小红书
> 人设规范：[persona_bible.md](persona_bible.md)
> 适用场景：发布会/大会/重大事件的精简报道，挑最有看点的 3-5 个点说清楚

## 与微信版的核心差异

| 维度 | 微信版 | 小红书版 |
|------|--------|---------|
| 目标 | 全面覆盖，信息不遗漏 | 挑最值得说的 3-5 个点 |
| 深度 | 每个产品都写 | 只写最有看点/最有争议的 |
| 字数 | 4000-8000 字 | 500-800 字 |
| 配图 | 产品截图为主 | 封面图 + 信息图为主 |
| 语气 | 有体验有吐槽 | 更直接、更感叹 |

## 标题公式（三选一，20字以内）

1. **速报型**：`{公司}刚刚放大招了！{最重磅的一个点}`
   - 例：Google 刚刚放大招了！Agent 电商三件套
2. **盘点型**：`{大会}最值得看的{N}个更新🔥`
   - 例：Google I/O 最值得看的 5 个更新🔥
3. **反差型**：`{公司}这次{出人意料的点}，没想到`
   - 例：Google 和 OpenAI 居然合作了，没想到

## 结构模板

### 开头：3 秒钩子（2-3 行）

```
{emoji} {一句话定性事件}

{最重磅的一个点抛出来}
```

**要点**：
- 第一句交代事件（"Google I/O 开完了"）
- 第二句抛最炸裂的点（"Agent 电商三件套才是真主角"）
- 不铺垫背景，直接进重点

### 主体：3-5 个要点（每个 2-3 行）

```
{emoji} {要点标题}
{1-2 句话说明}，{关键数据或评价}
```

### 选点原则

小红书版只选**最有看点的 3-5 个**，不是全量覆盖：

| 选点标准 | 示例 |
|---------|------|
| 最重磅的新功能 | Agent 电商、Universal Cart |
| 最有争议的 | 跟 OpenAI 合作、价格调整 |
| 最反差的 | Google 居然做了 X |
| 最实用的 | Ask Maps、Docs Live |
| 最好玩的 | Android XR 眼镜演示 |

**不要选的**：
- 纯技术参数更新（跑分提升 X%）
- 跟竞品差不多的功能
- 地区限制严重的功能

### 推荐要点结构

```
🔥 {最重磅的新功能}
{一句话说清}，{为什么重要}。

💡 {最有趣的更新}
{具体功能}，{体验评价}。

📊 {最实用的改进}
{对用户意味着什么}。

🤝 {最有故事感的细节}
{反差/意外/合作}。

💰 {价格/订阅变化}
{具体变化}，{你的评价}。
```

### 结尾：互动引导

```
💬 你对这次{大会}最感兴趣的是哪个？
📌 收藏这篇，回头慢慢看
```

## 语言风格

| 原则 | 要求 |
|------|------|
| 直接 | 不铺垫，直接说重点 |
| 感叹 | "真的猛"、"没想到"、"有点拉" |
| Emoji | 每个要点 1 个功能性 Emoji |
| 数据 | 关键数字必须有 |
| 短句 | 每句不超过 25 字 |

### 微信 → 小红书简化对照

| 微信版写法 | 小红书版写法 |
|-----------|------------|
| "Gemini 3.5 Flash 在 Terminal-Bench 2.1 编码测试拿了 76.2%，3.1 Pro 只有 70.3%" | "新 Flash 模型编码能力直接碾压上一代 Pro" |
| "Google 宣布 OpenAI、Kakao、ElevenLabs 也加入了 SynthID" | "Google 和 OpenAI 居然合作了🤯 一起搞 AI 内容水印" |
| "Universal Cart 今年夏天美国上线，先在 Search 和 Gemini App 里" | "跨商家智能购物车来了🛒 看 YouTube 也能加购物车" |
| 500 字的 Antigravity 2.0 介绍 | "Google 的编程 Agent 大升级，93 个子 Agent 并行跑，12 小时造了个 OS" |
| 价格体系详细对比 | "Ultra 从 $250 降到 $200，新增 $100 档" |

## 封面图建议

- 尺寸：1080×1440px（3:4 竖版）
- 主标题：公司名 + 大会名 + 核心看点（60-80px 粗体）
  - 例："Google I/O / 20+ 新功能 / Agent 才是主角"
- 副标题：最炸裂的一个点（30-40px）
  - 例："Google 和 OpenAI 居然合作了"
- 背景：大会主视觉/产品截图
- 配色：产品品牌色

## 配图清单（小红书版）

- [ ] 封面图（决定点击率）
- [ ] 核心产品截图 1-2 张
- [ ] 信息图/对比图（价格变化、功能对比）
- [ ] 演示视频关键帧（如果有好玩的）
- 建议 4-6 张图

## 禁止使用

- 超过 30 字的单句
- 连续 3 行没有 Emoji 或加粗
- "让我们来看看"、"今天给大家分享"
- 纯技术参数堆砌（要有看点/评价）
- 全量覆盖（只挑最有看点的 3-5 个）
