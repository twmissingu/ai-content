# 账号注册指引

系统需要你拥有以下社交平台账号，才能在分发环节把内容推到草稿箱。

---

## 注册顺序（按优先级）

### 1️⃣ 小红书（必开，首选平台）

**用途：** 内容分发主阵地，AiToEarn 任务最多。

**步骤：**
1. 下载「小红书」App
2. 手机号注册
3. 设置昵称、头像、个人简介
4. 完善资料 → 选「科技/数码」领域

**门槛：** 发文 0 粉即可，接广告任务需要 100-500 粉。

---

### 2️⃣ 微信公众号（必开）

**用途：** 深度长文分发，头条号/百家号内容由此同步。

**步骤：**
1. 打开 [https://mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 点击「立即注册」→ 选择「订阅号」（个人用订阅号）
3. 用邮箱注册（未注册过公众号的邮箱）
4. 激活邮箱 → 选择「个人」主体类型
5. 填写身份证信息进行实名认证
6. 设置公众号名称、简介、头像

**门槛：** 注册即用，无需粉丝。日推 1 篇。

---

### 3️⃣ 抖音（推荐开）

**用途：** 短视频分发，AiToEarn 分发通道支持。

**步骤：**
1. 下载「抖音」App
2. 手机号注册
3. 设置资料

**门槛：** 发文 0 粉即可。

---

### 4️⃣ 视频号（推荐开）

**用途：** 视频内容分发。

**步骤：**
1. 打开微信 → 发现 → 视频号
2. 点击右上角人像图标 → 创建视频号
3. 设置名称和简介

**门槛：** 有微信即可开通。

---

### 5️⃣ 头条号（可选，Phase 3 才需要）

**用途：** 同步公众号内容。

**步骤：**
1. 打开 [https://mp.toutiao.com](https://mp.toutiao.com)
2. 手机号注册
3. 实名认证

**门槛：** 注册即用。

---

### 6️⃣ 百家号（可选）

**用途：** 同步公众号内容。

**步骤：**
1. 打开 [https://baijiahao.baidu.com](https://baijiahao.baidu.com)
2. 百度账号登录 → 注册百家号
3. 实名认证

---

## 注册完成后

把注册好的账号连接到 AiToEarn：

1. 登录 [https://aitoearn.cn](https://aitoearn.cn)
2. 点击左侧「设置」→「API Key」
3. 确认已有 API Key（已配置到 Hermes）
4. 点击「Add Channels」→ 选择平台 → 扫码授权

连接成功后，系统的 Publisher Agent 就能把文章推到对应平台的草稿箱了。
