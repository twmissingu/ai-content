# 稿定 · 知识库

## 目录结构

| 目录 | 用途 |
|------|------|
| `topics/` | 选题索引（按日期） |
| `viral/` | 爆款库（阅读量前20%的文章模式） |
| `history/` | 历史文章归档（按日期分目录） |
| `strategy/` | 策略库（趋势分析、选题方向） |
| `materials/` | 素材库（手动收藏的内容） |

## 使用说明

- Knowledge Agent 自动维护本目录
- 历史文章按 `kb/history/YYYY-MM-DD/` 归档
- 支持 Markdown + wikilink 语法
- Dashboard 知识库 Tab 支持全文搜索
